# AniTrak - WORK IN PROGRESS


## A Anime tracking application, built for macOS 26

small application used to keep track of anime I am watching
built with SwiftUI

Adapted from AnimeSwiftForm


## Planned features

- track anime, and each episode watched, anime info will be pulled automatically from MAL with an option to manually input fields

- Can add anime watched, but not nececarily tracking. will pull data from MAL, can convert to tracked anime

- "Wishlisht" anime, can convert to tracked anime or leave as is

- Rich Liquid Glass UI

- WebApp Interopabilty, Will have local and web accounts for future web app

- Statistics galore: Time watched, episodes watched, etc...

No promises
